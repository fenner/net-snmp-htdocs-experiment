Release Schedule

{{ page.title }}

The Net-SNMP project releases patch releases off of a git branch named V5-9-patches (for example) and full releases off of the master branch. All changes to the patches branch are regularly rolled into the master branch.

Any code changes implementing new features or requiring API updates, etc, must only go into the master branch. Patches branches are for bug fixes only.
